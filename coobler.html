Coobler

Cooking in rural Ethiopia has some related problems that could be solved by the implementation of a different way of cooking such as using the sun as energy. The following project aims to make cooking a safe activity with the creation of a solar cooking device specially designed for the Ethiopian market.

 

This project was built in collaboration with the NGO Stitching Solar Cooking Nederland.

The CooBLER aims for safety during cooking, aims for sustainability, and aims for better quality of life. The CooBLER consists of a foldable wooden table, which can be used in different settings as for example, based on the floor, with different heights, or hanged on the wall. This table contains a drawer in which the needed materials for cooking, using the sun as energy, are stored.
